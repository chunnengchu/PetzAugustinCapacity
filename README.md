# PetzAugustin
A demonstration of two approaches for computing Petz-Augustin capacity
## Run 
```
$ python expr_Capacity.py
```

## References
We use the QuTip package.\
J.R. Johansson, P.D. Nation and F. Nori, QuTiP: An open-source Python framework for the dynamics of open quantum systems, *Comp. Phys. Comm.*, 2012 ([link](https://doi.org/10.1016/j.cpc.2012.02.021))\
Our first approach is based on Nesterov's universal fast gradient method (FGM)
Y. Nesterov, Universal gradient methods for convex optimization problems, *Math. Program.*, 2015
